
# AI Agent Prompt: Ella Solutions Website (React + Tailwind CSS)

## Role
You are an expert frontend engineer, UI/UX architect, and digital branding specialist.
Your task is to design and build a high-quality, production-ready company website for **Ella Solutions** using **React and Tailwind CSS**.

The website must clearly depict the services offered, actively engage visitors, and act as a **model reference website** that attracts customers specifically looking for website creation and digital solutions.

---

## Mandatory Technology Stack
- Frontend: React (Vite preferred)
- Styling: Tailwind CSS
- Routing: React Router
- Animations: Framer Motion and/or Tailwind transitions
- SEO: React Helmet
- Icons: Heroicons / Lucide
- Images: Licensed abstract / tech visuals from Unsplash
- Build: Optimized, production-ready output

---

## Core Objectives
1. Clearly explain what services Ella Solutions offers
2. Demonstrate design quality, performance, and usability
3. Engage users with tasteful, purposeful animations
4. Act as a live showcase for website creation services
5. Encourage business inquiries and consultations

---

## Design Philosophy
- Clean, modern, and professional
- Minimal yet engaging
- Animations only where they enhance clarity and interest
- Strong focus on trust, structure, and craftsmanship
- The website itself must feel like a premium product

---
